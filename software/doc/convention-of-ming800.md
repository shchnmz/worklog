# 明日系统信息输入约定

#### 背景
* 临时转班系统以及正在开发的教学管理后台需要从明日系统导入数据（年级，班级，学生...）
* 规范的信息输入可以确保数据导入的自动化
* 极大的减少人力输入，检查等工作量

#### 具体约定
* 年级命名
    * `年级` +`（校区）`。例子：`幼中（威宁）`
    * 不要出现`新`
    * 括号统一为**中文全角括号**

* 班级命名
   
    | 类型 | 命名规范 | 举例 |
    | ----| ---- | ---- |
    | 老班 | `班级名称` + `（校区）` | `三年级1（威宁）` |
    | 新班 | `2位年数` + `学期（秋／春）`+ `新`+ `班级名称` + `（校区）` | `17秋新基二三1 (威宁)` |

* 班级上课时间
  * 使用**自定义**时间。例如：`星期二 16:25-17:55` 
  * 严格和实际时间**保持一致**
