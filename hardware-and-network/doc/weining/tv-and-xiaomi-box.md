# 威宁校区电视机以及小米盒子

## 康佳电视机类型

| 康佳电视类型 | 第一次按下“输入选择”键出现的菜单 | 选择输入信号的 方法 |
| :--: | :--: | :--: |
| A | 出现横向菜单 | 多次按“输入选择”键 ，切换输入信号，停止后，系统会自动确定当前信号源（转圈） |
| B | 出现纵向菜单 | 按”音量（减小）“ / ”音量（增大）“，切换输入信号，再次按”输入选择“确认 |
| C | 开机很慢，按下“输入选择”没反应，等5-10分钟后，再按“输入选择”出现横向菜单 | 同康佳电视A类型 |
