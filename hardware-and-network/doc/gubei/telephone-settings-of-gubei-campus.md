# 古北校区电话相关配置信息

#### 电信Modem接口
* 语音1：接1117号码
* 语音2：接1115号码

#### 分机

| 房间 | 内部分机 | 外线号码 | 备注 |
| :----- | :--------- | :------- | :----- |
| 1楼前台 | 8001 | 1115 | |
| 1楼会议室 | 8002 | 1115 | 地面2根网线接会议桌，其中1根有“电话”标签。其中蓝白，蓝色一对线是电话线，已经接入程控交换机 |
| 1楼办公室 | 8005 | 1117 | 外部拨1117可以直达 |

#### 外线转内线
* 先按星号键(`*`)，然后加分机号码

#### 内线转内线
* 先按闪断键`R`，然后加分机号码

#### 内线拨外线
* 直接拨外线号码
